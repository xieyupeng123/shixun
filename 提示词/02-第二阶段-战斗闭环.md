### 任务2：Weapon武器精灵（weapon.py）

Weapon继承pygame.sprite.Sprite：
- 从player.status提取方向（split('_')[0]）
- 加载对应方向的武器图片：`../graphics/weapons/{weapon}/{direction}.png`
- 根据方向放置在玩家对应侧：
  - right → midleft = player.rect.midright + (0, 16)
  - left → midright = player.rect.midleft + (0, 16)
  - down → midtop = player.rect.midbottom + (-10, 0)
  - up → midbottom = player.rect.midtop + (-10, 0)
- sprite_type = 'weapon'

### 任务3：MagicPlayer魔法系统（magic.py）

MagicPlayer类：
- 治疗魔法heal(player, strength, cost, groups)：
  - 检查energy >= cost
  - 播放heal音效
  - player.health += strength（不超过max）
  - 扣除energy
  - 创建'aura'和'heal'粒子

- 火焰魔法flame(player, cost, groups)：
  - 检查energy >= cost
  - 扣除energy，播放flame音效
  - 根据player朝向，在前方1-5格位置创建'flame'粒子
  - 每个粒子位置加随机偏移（±TILESIZE/3）

### 任务4：Enemy敌人类（enemy.py）

继承Entity，实现AI状态机：

**初始化**：
- 加载3组动画：idle/move/attack
- 读取monster_data配置：health, damage, speed, exp, attack_radius, notice_radius
- Boss特殊处理：is_boss标志，hitbox缩小(48×48)，phase属性着色
- A*寻路：path列表，500ms重算间隔，玩家换格时重算

**AI状态机**（get_status + actions）：

| 状态 | 条件 | 行为 |
|------|------|------|
| idle | distance > notice_radius | 原地播放idle动画 |
| move | distance ≤ notice_radius | A*寻路追踪玩家 |
| attack | distance ≤ attack_radius且can_attack | 造成伤害，400ms冷却 |
| hit | 被攻击后 | 300ms无敌闪烁+击退（方向取反×resistance） |
| death | health ≤ 0 | 播放死亡粒子+经验球+浮动文字 |

**伤害处理** get_damage(player, attack_type)：
- 检查vulnerable
- weapon攻击：health -= player.get_full_weapon_damage()
- magic攻击：health -= player.get_full_magic_damage()
- 设置vulnerable=False, hit_time=当前时间

**Boss回调**：
- on_boss_killed回调 → 通知BossManager._on_boss_killed

### 任务5：粒子特效系统（particles.py）

AnimationPlayer类管理所有粒子：

**粒子类型**（预加载动画帧）：
- 魔法：flame(火焰), aura(光环), heal(治疗)
- 攻击：claw(爪击), slash(斩击), sparkle(火花), leaf_attack(叶刃), thunder(雷电)
- 怪物死亡：squid(橙色烟雾), raccoon(浣熊), spirit(新星), bamboo(竹子)
- 草叶：leaf1-leaf6（含镜像版本）

**ParticleEffect**精灵：
- 帧动画播放，播放完毕自动kill
- animation_speed = 15

**MovingParticleEffect**（经验球追踪）：
- 从敌人位置飞向玩家位置
- speed=250像素/秒
- 到达或超时(1.5s)后消失

**FloatingText**（浮动文字）：
- 显示"+100 XP"等文字
- 上升+淡出效果
- duration=1.2秒

### 任务6：音效系统集成

在以下位置集成音效：
- 玩家攻击：weapon_attack_sound（sword.wav）
- 玩家受伤：无（通过粒子反馈）
- 敌人受击：hit_sound（hit.wav）
- 敌人死亡：death_sound（death.wav）
- 敌人攻击：attack_sound（根据怪物类型：slash/claw/fireball）
- 魔法释放：heal音效 / flame音效
- 无敌模式：invincible_sound（loop）
- 背景音乐：game_bgm.ogg（普通）/ boss_bgm.ogg（Boss区域）

所有音效通过ResourceManager.instance()缓存，失败静默跳过。

### 任务7：BossManager（level.py内）

BossManager类：
- spawn_boss(phase)：在固定位置生成Boss（raccoon），应用phase属性倍增
- _on_boss_killed(death_pos)：boss_kills++，判断是否触发胜利，否则spawn下一phase
- 三阶段：phase1(×1/无色) → phase2(×2/淡红) → phase3(×4/紫色)

### 任务8：BGM区域切换

MusicState类（单例状态）：
- state=0：普通BGM
- state=1：Boss BGM

Level._check_boss_music()：
- 一次性切换：玩家进入Bossnotice_radius时，永久切换到Boss BGM
- 无敌模式下只设置状态，不立即播放
- 退出无敌时恢复当前BGM

### 任务9：无敌模式

Player空格键切换：
- 检查_invincible_keys_released防止连触
- 开启：停止当前BGM，播放invincible_sound(loop)
- 关闭：停止invincible_sound，恢复当前BGM（根据MusicState判断）
- 视觉：animate中叠加金色半透明层（BLEND_RGBA_ADD）
- 属性：速度×2，攻击×2（在get_full_weapon_damage/magic_damage中应用）
