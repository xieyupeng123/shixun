# 🎮 Stardew Valley Action X

> 一款轻量级 2D 动作冒险游戏，使用 Python + Pygame 构建

---

## 📖 项目简介

**Stardew Valley Action X** 是一款俯视角 2D 动作 RPG，玩家在像素风格地图中探索、战斗。采用模块化架构设计，支持逐步扩展。

该项目为课程实训项目，从零开始迭代开发。

---

## 🚀 快速开始

### 环境要求
- Python 3.10+
- Pygame 2.x

### 安装运行
```bash
# 安装依赖
pip install pygame

# 运行游戏
cd code
python main.py
```

---

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| `W` `A` `S` `D` / 方向键 | 移动角色 |
| `SPACE` | 挥剑攻击 |
| `ENTER` | 开始游戏 |
| `R` | 死亡后重新开始 |
| `ESC` | 退出游戏 |

## 🏗️ 项目架构

```
code/
├── main.py           # 入口：游戏状态机 (开始页/游戏中/死亡页)
├── settings.py       # 全局配置：窗口/地图/武器/怪物数据
│
├── entity.py         # Entity 基类：移动 + 碰撞检测
├── player.py         # 玩家：移动/攻击/HP/受伤/动画
├── enemy.py          # 敌人：AI状态机 (idle/move/attack)
│
├── level.py          # 关卡：地图加载 + 摄像机跟随 + 战斗集成
├── tile.py           # 地图瓦片
├── weapon.py         # 武器精灵
│
├── combat.py         # 战斗计算：伤害/击退/暴击
├── ui.py             # HUD 界面：HP条 + EXP显示
├── particles.py      # 粒子特效：受击反馈
├── items.py          # 物品系统：物品类 + 背包 (骨架)
├── save_manager.py   # 存档系统：JSON 存/读档 (骨架)
│
└── support.py        # 工具函数：路径/CSV/文件夹加载
```

### 架构图

```
main.py (状态机: start / game / death)
  ├── level.py          ← 关卡管理
  │     ├── player.py   ← 玩家

  │     ├── weapon.py   ← 武器精灵
  │     ├── tile.py     ← 地图瓦片
  │     └── particles.py← 粒子特效
  ├── ui.py             ← HUD 界面
  ├── combat.py         ← 战斗计算
  ├── items.py          ← 物品背包
  ├── save_manager.py   ← 存档系统
  └── settings.py       ← 全局配置
```

## 🛠️ 技术栈

- **语言**: Python 3.12
- **引擎**: Pygame 2.6
- **地图**: CSV 瓦片地图
- **碰撞**: AABB 矩形碰撞 + 分离轴
- **渲染**: Y轴深度排序

---

## 📝 开发笔记

- 敌人 AI 采用简单状态机：`idle` → `move`(检测到玩家) → `attack`(进入攻击范围)
- 精灵动画使用 Delta Time 帧索引，保证不同帧率下动画速度一致
- 摄像机通过偏移绘制实现，无需移动精灵实际坐标
- 武器精灵程序化生成，无需外部资源文件
- 所有新增模块遵循 Entity 基类继承模式，保持代码一致性
